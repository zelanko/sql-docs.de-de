# <a name="report-data-f1-help"></a>Hilfe (F1) zu Berichtsdaten
  Die Themen in diesem Abschnitt enthalten Informationen zu den Optionen, die Sie für Datenquellen, freigegebene Datenquellen und Datasets festlegen können, sowie Informationen zu den im Berichts-Designer verfügbaren Abfrage-Designern.  
  
 Um innerhalb des Dialogfelds **Berichtseigenschaften** auf diese Themen zuzugreifen, klicken Sie auf **Hilfe**.
