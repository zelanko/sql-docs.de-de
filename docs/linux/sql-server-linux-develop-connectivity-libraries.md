# <a name="connectivity-libraries-and-frameworks-for-microsoft-sql-server"></a>Verbindungsbibliotheken und-Frameworks für Microsoft SQL Server

[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-linuxonly](../includes/appliesto-ss-xxxx-xxxx-xxx-md-linuxonly.md)]

Sehen Sie sich die [Tutorials zu ersten Schritten](http://aka.ms/sqldev) schnell erste Schritte mit Programmiersprachen wie c#, Java, Node.js, PHP und Python, und Erstellen einer app mithilfe von SQL Server unter Linux oder Windows oder mit Docker unter MacOS.
