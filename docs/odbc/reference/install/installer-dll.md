# <a name="installer-dll"></a>Installationsprogramm-DLL
> [!NOTE]  
>  Ab Windows XP und Windows Server 2003, ist ODBC in das Windows-Betriebssystem enthalten. Sie sollten nur explizit ODBC in früheren Versionen von Windows installieren.  
  
 Das Installationsprogramm-DLL enthält Funktionen zum Installieren und Entfernen von ODBC-Komponenten, Informationen zu diesen Komponenten in der Registrierung zu verwalten und Verwalten von Informationen zu den Datenquellen in der Registrierung. Es wird von Microsoft geschrieben und verteilt werden kann. Eine vollständige Beschreibung der Funktionen in die Installationsprogramm-DLL, finden Sie unter [Installationsprogramm-DLL-API-Referenz](../../../odbc/reference/syntax/installer-dll-api-reference-function.md).
