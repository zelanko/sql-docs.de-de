# <a name="using-data-buffers"></a>Verwenden von Datenpuffern
Datenpuffer werden durch drei Arten von Informationen beschrieben: ihre Typ, Adresse und Byte-Länge. Wenn eine Funktion benötigt eine der folgenden Angaben, und es nicht bereits bekannt ist, hat er ein Argument mit dem die Anwendung übergibt.
